# Student Management System

A backend application built using Spring Boot for managing student records.

## Features

* Create student records
* View student information
* Update student details
* Delete student records
* RESTful API architecture

## Technologies Used

* Java
* Spring Boot
* Maven
* MySQL
* Spring Data JPA

## Project Structure

```text
src/
├── controller
├── service
├── repository
├── entity
└── resources
```
